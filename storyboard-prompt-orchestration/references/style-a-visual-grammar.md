## Colored-Pencil Semantic Color Defaults

| Color | Semantic Role | Use For |
|-------|--------------|---------|
| **Blue** | Structure | Layout, containers, frames, grids, hierarchy |
| **Red** | Interaction | Movement, direction, emphasis, critical paths, CTAs |
| **Yellow** | Highlight | Selected areas, attention zones, active states |
| **Green** | State | Confirmation, success, grouping, completed steps |
| **Orange** | Callout | Energy, transitions, annotations, warnings |

### Color Rules
- Use 2-3 colors per zone, not all 5 everywhere
- Blue is the dominant structural color (appears most)
- Red for the most important flow/interaction paths
- Yellow sparingly — only for true focal points
- Green and orange as accents
- Never use saturated flat fills — always pencil texture

## Anti-Polish Discipline

These rules prevent drift toward production/mockup aesthetics:

1. **Lines must wobble.** No perfectly straight edges. Hand-drawn means hand-drawn.
2. **Shapes must be imperfect.** Rectangles slightly uneven. Circles slightly elliptical.
3. **Text must look handwritten.** Block letters, slightly uneven baseline, not typeset.
4. **No gradients.** Colored-pencil texture only. Crosshatch or light fill, never smooth gradient.
5. **No drop shadows.** Depth comes from layering and overlap, not shadow effects.
6. **No icon libraries.** Simple hand-drawn symbols only (gear, star, arrow, etc.)
7. **Construction marks visible.** Light guide lines, alignment marks, erased-but-visible strokes.
8. **Asymmetric layout.** Zones should feel organically placed, not pixel-grid-snapped.
9. **Mixed scale.** Some elements larger for emphasis, not uniform sizing.
10. **White space is breathing room.** Not every zone edge-to-edge filled.

## Sketchbook Aesthetic Requirements

The final prompt must produce something that feels like:
- A page torn from a designer's Moleskine
- Pinned to a corkboard in a design studio
- Photographed on a desk with coffee stains nearby
- NOT: a Figma screenshot, a Dribbble shot, a Behance presentation

### Texture Signals
- Paper grain visible
- Pencil pressure variation (light sketch lines vs. bold outlines)
- Occasional smudge or correction mark
- Layered strokes showing iteration
- Organic edges where color bleeds slightly
